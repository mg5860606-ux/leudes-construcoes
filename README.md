## 📌 Sobre o Projeto

O website da **Leudes Construções** é uma landing page moderna, institucional e de alta conversão, desenvolvida para apresentar serviços de construção civil, empreitada e reformas residenciais e comerciais.

Com foco em **transparência, profissionalismo e facilidade de contato**, a página oferece uma experiência impecável tanto para dispositivos móveis quanto para computadores, permitindo que clientes solicitem orçamentos de forma rápida e direta.

---

## 🚀 Funcionalidades Principais

- 📱 **Design 100% Responsivo**: Layout otimizado para celulares, tablets e desktops.
- ⚡ **Navegação Fluida**: Cabeçalho fixo (*sticky*) com transição suave (*smooth scroll*).
- 🛠️ **Catálogo de Serviços**: Apresentação clara de fundações, alvenaria, cobertura, acabamentos e reformas em geral.
- 👷 **Apresentação Profissional**: Seção institucional destacando a reputação, valores e uniforme oficial da empresa.
- 💬 **Integração com WhatsApp**: Botões diretos para solicitação de orçamento com mensagens pré-configuradas.
- ⭐ **Depoimentos e Prova Social**: Avaliações de clientes para gerar autoridade e confiança.
- 📊 **Calculadora / Formulário Dinâmico**: Facilita o envio de detalhes sobre a obra pretendida.

## 💻 Como Executar o Projeto Localmente

Como o projeto utiliza tecnologias Web nativas (HTML, CSS e JavaScript), não é necessária a instalação de dependências ou build servers.

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/mg5860606-ux/leudes-construcoes.git
   ```

2. **Acesse o diretório:**
   ```bash
   cd leudes-construcoes
   ```

3. **Abra no navegador:**
   - Basta dar um duplo clique no arquivo `index.html`, ou
   - Utilizar uma extensão como o **Live Server** no VS Code.
